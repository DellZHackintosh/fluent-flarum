### Fluent-Flarum

使用这款主题为你的 Flarum 换新颜。

本主题参考了 Microsoft Fluent Design System 进行设计，为您带来熟悉、柔和、亲切和流畅的感觉。

#### 覆盖的内容

按钮、切换开关、选项菜单、标签、边栏、模态窗口、信息栏、输入控件和复选框。



不确定是否与 Flarum Categories 兼容。
